## 已知問題與解決方案

### 問題 1: PDF.js Worker 加載失敗
- **原因**: 本地 node_modules 路徑無法通過 Vite 代理訪問
- **解決方案**: 使用 cdnjs CDN (v3.11.174)
- **狀態**: ✓ 已解決

### 問題 2: 檔案選擇按鈕無法開啟
- **原因**: input 元素在 Card 內部，事件冒泡被阻止
- **解決方案**: 將 input 移到組件頂部，添加多個事件監聽器
- **狀態**: ✓ 已解決
